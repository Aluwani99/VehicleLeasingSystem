# Vehicle Leasing System – Junior Dev Test

## Overview
ASP.NET MVC app to manage leased vehicles and their relationships with suppliers, branches, clients, and drivers.

## Tech
- ASP.NET MVC (.NET Framework 4.7.2)
- Entity Framework 6 (Code First)
- SQL Server Express
- Bootstrap

## Features
- Full CRUD for Suppliers, Branches, Clients, Drivers, Vehicles
- Manufacturer report by Supplier, Branch, Client
- Sample data pre-seeded
- ERD and DB backup included
